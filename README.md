# PanjoCoin (PNJC)

**PanjoCoin (PNJC)** is an ERC-20 utility token on the Polygon network with a fixed supply of 1 trillion tokens.

## Key Features
- ERC-20 standard: transfers and permissions
- Fixed supply: 1,000,000,000,000 PNJC
- Token burning: burn() and burnFrom()
- Admin functions: renounceAdmin() for decentralization
- Auditable events: TokensBurned and AdminRenounced

## Purpose
Support for the **ClownCare** project through the token creator's family foundation, **Charity Fund SmileDonate** ([www.smiledonate.com](http://www.smiledonate.com))
